# lp-emprender
Landing page Emprender


##  Requirements

```
Este landing page necesita "correr" con un  preprocesador, en este caso se está usando sass.
Para correr localmente, use npm.

1- Instalar Sass

Abra su terminal, ubicado en la carpeta “src”.
Ejecute el comando: sass —watch sass:css
Una vez ejecutado el comando anterior, Sass procederá a generar el archivo CSS compilado.
Encontrará el archivo CSS compilado, en la carpeta “src”. Sass habrá creado automáticamente una carpeta llamada “css”, donde encontrará el archivo con extensión .css y otro archivo con extensión .css.map. Por favor, NO altere este último archivo.


2- Instalar el sitio localmente (utilizando nvm).


Verificar la versión de Node.js:
Nota: Para poder instalar este sitio de forma local, es necesario contar con una versión de Node.js superior a la 8.x

Para verificar la versión de Node.js que está utilizando, corra el comando: node -v


Utilizando npm:
Abra su terminal, (ubicado en la carpeta “lp-emprender”).
Ejecute el comando: npm install
Finalmente, una vez que se instalen todas las dependencias (esto puede tomar su tiempo), ejecute el comando: npm start.


Para más información, consulte la documentación respectiva:

Sass: https://sass-lang.com 

Node.js: https://www.npm.js.com

```
